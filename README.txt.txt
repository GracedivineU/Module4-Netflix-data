Netflix Shows and Movies Data Analysis


This project fulfills the requirements of:

1. Data preparation
2. Data cleaning
3. Data exploration and statistical analysis
4. Data visualization using Python
5. Chart creation using R


Dataset Used

File Name: Netflix_shows_movies.csv

How to Run the Project

Requirements
Install the required libraries:

Python Libraries:
pip install pandas matplotlib seaborn

R Libraries:
install.packages("ggplot2")
